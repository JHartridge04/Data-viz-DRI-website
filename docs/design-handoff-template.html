Library Visualization

Design Handoff Template — Fill this out in Canva, Figma, or any design tool

Overview

The Library visualization has three views that users move through. Each view has designable areas where your team's visuals will replace the current AI-generated placeholder styling. This document shows every designable area, its dimensions, and what to export.

How this works: Your team designs the visual assets (backgrounds, textures, color palettes). The developer then plugs them into the code by updating CSS values and image references. The interactive functionality (clicking, searching, filtering) stays the same — only the look changes.

How to Design in Canva

For backgrounds (A1, B1): Create a custom-size design at 1920×1080. Design your library scene or texture. Export as PNG.

For the shelf end-cap (A2): Create a custom-size design at 300×360. Design the shelf face with any shelf lines, texture, or styling. Leave a centered area empty for the genre label text (the code overlays it). Export as PNG with transparent background.

For the shelf texture (B2): Create a custom-size design at 2200×320. Design the wood or surface texture. Export as PNG.

For style decisions: Create a single "style guide" page in Canva showing your chosen colors, fonts, and the overall mood. This doesn't need to be pixel-perfect — it's a reference for the developer.
